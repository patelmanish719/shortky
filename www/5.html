MS Excel Shortcut Keys
Ctrl + A Select all contents of a worksheet
Ctrl + B Bold all cells in the highlighted section
Ctrl + C Copy all cells in the highlighted section
Ctrl + D Fill down
Ctrl + F Search current sheet
Ctrl + G Go to a certain area
Ctrl + H Find and replace
Ctrl + I Puts italics on all cells in the highlighted section
Ctrl + K Insert a hyperlink
Ctrl + N Create a new workbook
Ctrl + O Open a workbook
Ctrl + P Print the current sheet
Ctrl + R Fill right
Ctrl + S Save the open worksheet
Ctrl + U Underline all cells in the highlighted section
Ctrl + V Paste everything copied onto the clipboard
Ctrl + W Close the current workbook
Ctrl + X Cut all cells in the highlighted section
Ctrl + Y Repeat the last entry
Ctrl + Z Undo the last action
Ctrl + 1 Change the format of the selected cells
Ctrl + 2 Bold all cells in the highlighted section
Ctrl + 3 Put italics all cells in the highlighted section
Ctrl + 4 Underline all cells in highlighted section
Ctrl + 5 Put a strikethrough all cells in the highlighted section
Ctrl + 6 Show or hide objects
Ctrl + 7 Show or hide the toolbar
Ctrl + 8 Toggle the outline symbols
Ctrl + 9 Hide rows
Ctrl + 0 Hide columns
Ctrl + Shift + : Enter the current time
Ctrl + ; Enter the current date
Ctrl +` Change between displaying cell values or formulas in the worksheet
Ctrl + ' Copy a formula from the cell above
Ctrl + Shift + " Copy value from cell above
Ctrl + - Delete the selected column or row
Ctrl + Shift + = Insert a new column or row
Ctrl + Shift + ~ Switche between showing Excel formulas or their values in cells
Ctrl + Shift + ! Apply comma formatting
Ctrl + Shift + $ Apply currency formatting
Ctrl + Shift + # Applies date formatting
Ctrl + Shift + % Apply percentage formatting
Ctrl + Shift + ^ Apply exponential formatting
Ctrl + Shift + * Select the current region around the active cell
Ctrl + Shift + & Place border around selected cells
Ctrl + Shift + _ Remove the border
Ctrl + + Insert
Ctrl + - Delete
Ctrl + Shift + ( Unhide rows
Ctrl + Shift + ) Unhide columns
Ctrl + / Select the array containing the active cell
Ctrl + \ Select the cells that have a static value or don’t match the formula in the active cell
Ctrl + [ Select all cells referenced by formulas in the highlighted section
Ctrl + ] Select cells that contain formulas that reference the active cell
Ctrl + Shift + { Select all cells directly or indirectly referenced by formulas in the highlighted section
Ctrl + Shift + } Select cells which contain formulas that directly or indirectly reference the active cell
Ctrl + Shift + | Select the cells within a column that don’t match the formula or static value in the active cell
Ctrl + Enter Fill the selected cells with the current entry
Ctrl + Spacebar Select the entire column
Ctrl + Shift + Spacebar Select the entire worksheet
Ctrl + Home Move to cell A1
Ctrl + End Move to last cell on worksheet
Ctrl + Tab Move between Two or more open Excel files
Ctrl + Shift + Tab Activate the previous workbook
Ctrl + Shift + A Insert argument names into a formula
Ctrl + Shift + F Open the for fonts
Ctrl + Shift + O Select all of the cells that contain comments
Ctrl + Shift + P Open the drop-down menu for point size
Shift + Insert Paste what is stored on the clipboard
Shift + Page Up In a single column, highlights all cells above that are selected
Shift + Page Down In a single column, highlights all cells above that are selected
Shift + Home Highlight all text to the left of the cursor
Shift + End Highlight all text to the right of the cursor
Shift + Up Arrow Extend the highlighted area up one cell
Shift + Down Arrow Extend the highlighted area down one cell
Shift + Left Arrow Extend the highlighted area left one character
Shift + Right Arrow Extend the highlighted area right one character
Alt + Tab Cycle through applications
Alt + Spacebar Open the system menu
Alt + Backspace Undo
Alt + Enter While typing text in a cell, pressing Alt+Enter will move to the next line, allowing for multiple lines of text in one cell
Alt + = Create a formula to sum all of the above cells
Alt + ' Allow formatting on a dialog box
F1 Open the help menu
F2 Edit the selected cell
F3 paste names
F4 Repeat last action For example, if you changed the color of text in another cell, pressing F4 will change the text in cell to the same color
F5 Go to a specific cell For example, C6
F6 Move to the next pane
F7 Spell check selected text or document
F8 Enter Extend Mode
F9 Recalculate every workbook
F10 Activate the menu bar
F11 Create a chart from selected data
F12 Save As option
